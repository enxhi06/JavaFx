# Papa's Pizza Order Builder

A desktop application built using Java and JavaFX that allows users to customize and place pizza orders through an intuitive graphical user interface (GUI). The application dynamically processes user inputs to calculate totals and generate a complete order receipt.

![Pizza Order UI]preview.png
## ✨ Features
* **Customer Information:** Captures customer first and last names for order tracking.
* **Size Selection:** Dynamic dropdown selection for choosing pizza sizes.
* **Crust Customization:** Radio button selection limiting users to one crust type (Hand-tossed, Thin-crust, or Deep-dish).
* **Topping Selection:** Multiple checkbox selections for toppings (Pepperoni, Sausage, Onion, Mushroom) with real-time price adjustments ($1.25 per additional topping).
* **Receipt Generation:** Generates a detailed breakdown of the selected items, base price, topping additions, and the final total cost upon placing the order.

## 🛠️ Tech Stack
* **Language:** Java
* **Framework:** JavaFX (GUI)
* **IDE:** Eclipse
